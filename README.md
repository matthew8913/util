# Утилита для обработки текстовых файлов

## Необходимое окружение

- Java 17

## Инструкция по запуску

1. Скачайте `util.jar`.
2. Откройте командную строку.
3. Из папки с файлом запустите программу с помощью следующей команды:
    ```
    java -jar util.jar
    ```

## Инструкция к использованию

Утилита поддерживает несколько команд:

- `-f`: Полная статистика по данным.
- `-s`: Стандартная статистика по данным.
- `-o some\path`: Выбор пути для сохранения.
- `-p prefix_`: Установка префикса для выходных файлов.
- `-a`: Режим добавления данных в файлы.

**Важно соблюдать структуру введенной строки:**
Сначала должны идти через пробел команды в формате, указанном выше, затем через пробел должны идти минимум один исходный файл (если их не будет – программа запросит).

**Пример корректной строки:**
```
-f -o some\path -a -p prefix_ some\path\file1.txt some\path\file2.txt
```

**Дублирование команд не является ошибкой, будет использована последняя введенная команда.**

**Пример:**
```
-f -s -o some\path1 -o some\path2 -a -p prefix_ some\path\file1.txt
```
При таком вводе программа будет выводить стандартную статистику, а файлы будут сохранены в `some\path2`.

**Важно!** Если возникают проблемы с кодировкой, рекомендуется использовать английский язык при вводе.

**Также важно!** Использовать нужно полные пути файлов и папок, с которыми вы работаете. Если вы копируете путь файла/директории из: **_Свойства(файла)->Безопасность_**, учтите, 
что в таком случае в начале скопированной строки будет невидимый символ, который нужно стереть и поставить пробел для корректной работы программы.
